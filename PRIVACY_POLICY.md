# Privacy Policy for Tab Auto-Grouper

Last updated: July 15, 2026

Tab Auto-Grouper does not collect, store, transmit, or sell any user data.

## What the extension does

The extension reads the URL of open browser tabs solely to determine each
tab's domain, for the purpose of grouping tabs from the same website together
using Chrome's built-in tab groups feature. This processing happens entirely
locally within the user's browser.

## Data collection

This extension does not collect personally identifiable information, browsing
history, financial information, health information, authentication
credentials, or any other user data. No data is transmitted to any server,
third party, or external service.

## Storage

The extension uses Chrome's local storage API (chrome.storage.sync) only to
save the user's preference for whether auto-grouping is enabled or disabled.
This preference is not shared with anyone.

## Third parties

This extension does not share any data with third parties.
